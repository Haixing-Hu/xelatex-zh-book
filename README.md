# Zh-Book

本项目提供了一个用于中文科技书籍的XeLaTeX模板。该模板严格遵循中文科技文献排版的相关国家标准规范，可帮助老师同学们方便地排版出漂亮的中文科技书籍。

## 项目主页

本项目主页地址为：[http://haixing-hu.github.io/xelatex-zh-book/][homepage]

[homepage]: http://haixing-hu.github.io/xelatex-zh-book/

## 功能特色

* 此模版用于生成符合中文科技文献排版的相关国家规范、行业标准的中文科技书籍；
* 使用XeLaTeX作为排版引擎，论文源码需要使用UTF-8编码；
* 自动生成一个默认的中文封面；
